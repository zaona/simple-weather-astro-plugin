# 简明天气同步器
参照 [该文档](https://plugindoc.astrobox.online) 进行开发

## 快应用包名
com.application.zaona.weather

## 命令
构建
```bash
pnpm run build
```

打包
```bash
pnpm run package
```

> 打包使用了 `zip` 命令
> https://stahlworks.com/dev/index.php?tool=zipunzip
